Getting Started with AusNimbus

This is an example application deployed on AusNimbus. For instructions on how to deploy your own application, start by reading the documentation or go straight to the dashboard.

Dashboard Documentation

How this works

  • Using the git source URL you provided, AusNimbus fetched your source code.
  • The platform then built it into a secure, fast and reliable container image.
  • The container image was then used to spin up a lightweight Docker container to serve this page.

Next steps

  • Head back to the dashboard and put in your own git source repository
  • Trigger a new, and the platform will automatically deploy your app
  • You can scale your application, push changes and add add-on components from the dashboard or command line tools.
